Detailed Description

template<typename T, typename Filter>
class Util::FilterEnumerator< T, Filter >

An enumerator that filters the elements of given inner enumerator. The filter parameter should be a callable object that accepts the wrapped enumerator's value type and returns a bool. When traversing the enumerator, it will call the predicate on each element and skip any where it returns false.
