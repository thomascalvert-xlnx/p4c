◆ shouldEmitTimestamp()

bool EBPF::EBPFPipeline::shouldEmitTimestamp ( ) const
inline

Returns whether the compiler should generate timestamp retrieved by bpf_ktime_get_ns().

This allows to avoid overhead introduced by bpf_ktime_get_ns(), if the timestamp field is not used within a pipeline.
